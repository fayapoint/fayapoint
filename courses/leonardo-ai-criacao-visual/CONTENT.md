# MÓDULO 1: FUNDAMENTOS — Conceitos de IA Visual

**MÓDULO 1: FUNDAMENTOS DO LEONARDO AI**
**AULA 2: CONCEITOS ESSENCIAIS DE IA GENERATIVA VISUAL**

**Carga Horária:** 2 horas
**Indicadores Trabalhados:** 1. Compreender como IA gera imagens; 2. Escolher parâmetros corretamente.

---

**CONTEÚDO PARA O ALUNO**

**1. Como Funciona a Geração de Imagens por IA**

Modelos como Stable Diffusion (base do Leonardo AI) são redes neurais treinadas em bilhões de pares imagem-texto. O processo: começa com ruído aleatório, progressivamente "limpa" o ruído guiado pelo seu prompt até formar uma imagem coerente. Este processo de difusão reversa é o que permite controle detalhado sobre o resultado.

**2. Parâmetros Essenciais**

- **Guidance Scale (CFG):** Quão fielmente segue o prompt. Baixo (1-5): criativo, inesperado. Alto (7-15): fiel ao prompt. Muito alto (15+): pode causar artefatos.
- **Steps:** Iterações de refinamento. Mais steps = mais qualidade mas mais lento. 25-40 é ideal.
- **Seed:** Número que determina a aleatoriedade inicial. Mesmo seed + mesmo prompt = mesma imagem.
- **Dimensions:** Resolução da imagem. Diferentes aspect ratios para diferentes usos.

**3. Negative Prompts**

Tão importante quanto o prompt principal: diga à IA o que NÃO incluir. Negativas comuns: "blurry, low quality, deformed, watermark, text, ugly, distorted, extra fingers, bad anatomy."

**4. Modelos Disponíveis**

- **Leonardo Diffusion XL:** Versátil, boa qualidade geral
- **Leonardo Vision XL:** Fotorealismo premium
- **Leonardo Anime XL:** Estilo anime/manga
- **DreamShaper:** Balanceado entre arte e realismo
- **SDXL:** Base Stable Diffusion, altamente customizável
- **Community Models:** Milhares de modelos fine-tuned pela comunidade

# MÓDULO 2: PROMPT ENGINEERING VISUAL — Composição

**MÓDULO 2: PROMPT ENGINEERING VISUAL**
**AULA 1: ESTRUTURA E COMPOSIÇÃO DE PROMPTS**

**Carga Horária:** 3 horas
**Indicadores Trabalhados:** 1. Escrever prompts visuais eficazes; 2. Controlar composição e estilo.

---

**CONTEÚDO PARA O ALUNO**

**1. Anatomia de um Prompt Visual**

Um prompt eficaz segue esta estrutura: [Sujeito] + [Ação/Pose] + [Ambiente] + [Iluminação] + [Estilo] + [Qualidade] + [Detalhes Técnicos]

Exemplo: "A confident businesswoman giving a presentation in a modern glass office, golden hour sunlight streaming through windows, corporate photography style, Canon EOS R5, 85mm lens, shallow depth of field, 8k ultra detailed"

**2. Vocabulário de Iluminação**

A iluminação define o mood da imagem: golden hour (dourado, quente), blue hour (azul, melancólico), studio lighting (neutro, profissional), dramatic lighting (contraste alto), backlight/rim light (silhueta), neon lighting (cyberpunk, moderno), volumetric lighting (raios de luz visíveis).

**3. Vocabulário de Câmera**

Simule efeitos fotográficos: wide angle (paisagem), telephoto (retrato, compressão), macro (detalhes), aerial/drone (vista aérea), fisheye (distorção), tilt-shift (miniatura), long exposure (movimento).

**4. Pesos e Ênfase**

No Leonardo AI, use parênteses para dar peso a termos: "(red hair:1.3)" enfatiza cabelo vermelho. Valores acima de 1.0 aumentam ênfase, abaixo diminuem. Combine para controle fino da composição.

# MÓDULO 3: MODELOS — Consistência e Branding

**MÓDULO 3: MODELOS E ESTILOS**
**AULA 2: CONSISTÊNCIA VISUAL E BRANDING**

**Carga Horária:** 2 horas
**Indicadores Trabalhados:** 1. Manter estilo consistente entre gerações; 2. Criar identidade visual com IA.

---

**CONTEÚDO PARA O ALUNO**

**1. Character Consistency**

Manter o mesmo personagem em múltiplas imagens: use Character Reference (upload foto de referência), mantenha seed fixo para variações sutis, documente prompts que funcionaram, use modelo fine-tuned com o personagem.

**2. Style Consistency para Marcas**

Crie um "style guide de IA": documente o prompt base (estilo, cores, iluminação), salve seeds que funcionam, use sempre o mesmo modelo, crie templates de prompt por tipo de conteúdo (post, banner, thumbnail).

**3. Paleta de Cores Controlada**

Especifique cores no prompt: "color palette: navy blue, gold, white" ou "muted earth tones" ou "vibrant neon colors." Para consistência de marca, inclua hex codes quando possível.

**4. Batch Generation**

Gere séries de imagens com estilo coeso: mesmo prompt base + variações no sujeito. Exemplo: "Flat icon of [ITEM], minimal style, gradient blue background, soft shadows" onde [ITEM] varia para cada ícone.

# MÓDULO 4: FERRAMENTAS — Canvas e Edição

**MÓDULO 4: FERRAMENTAS AVANÇADAS**
**AULA 1: AI CANVAS E EDIÇÃO**

**Carga Horária:** 2 horas
**Indicadores Trabalhados:** 1. Usar AI Canvas para edição; 2. Expandir e modificar imagens.

---

**CONTEÚDO PARA O ALUNO**

**1. AI Canvas — Editor Integrado**

O Canvas combina geração com edição: inpainting (substituir partes da imagem), outpainting (expandir além das bordas), sketch-to-image (esboço vira arte), text-to-image (gerar dentro do canvas), camadas (trabalhar em múltiplas camadas).

**2. Inpainting — Edição Cirúrgica**

Selecione uma área da imagem e descreva o que quer lá: mudar roupas, trocar backgrounds, adicionar objetos, remover elementos indesejados. Dica: selecione área generosa ao redor do que quer mudar para transição suave.

**3. Outpainting — Expandir Imagens**

Pegue uma imagem e expanda em qualquer direção. O modelo gera conteúdo coerente que continua a composição original. Ideal para: adaptar aspect ratios, criar panoramas, adicionar espaço para texto.

**4. Realtime Canvas**

Geração em tempo real enquanto você desenha/escreve. Ideal para brainstorming visual: faça um esboço grosseiro e veja a IA transformar em arte instantaneamente.

# MÓDULO 4: FERRAMENTAS — Motion e Upscale

**MÓDULO 4: FERRAMENTAS AVANÇADAS**
**AULA 2: MOTION, UPSCALE E 3D TEXTURES**

**Carga Horária:** 2 horas
**Indicadores Trabalhados:** 1. Animar imagens estáticas; 2. Aumentar resolução e gerar texturas.

---

**CONTEÚDO PARA O ALUNO**

**1. Motion — Animação de Imagens**

Transforme imagens estáticas em vídeos curtos: selecione imagem gerada, configure intensidade de movimento, duração (4 segundos), estilo de câmera (zoom, pan, orbit). Ideal para posts de redes sociais, thumbnails animados, apresentações.

**2. Universal Upscaler**

Aumente resolução em até 4x sem perda de qualidade: upload de qualquer imagem (não apenas geradas no Leonardo), configurar nível de upscale e criatividade. Criatividade alta adiciona detalhes; baixa preserva original.

**3. Texture Generation**

Gere texturas seamless para 3D e design: madeira, metal, tecido, pedra, sci-fi surfaces. Exporta tileable para uso em Unity, Unreal Engine, Blender. Ideal para game developers e designers 3D.

**4. Remove Background**

Remoção automática de fundo: resultado clean para uso em composições, mockups, e-commerce. Combine com outpainting para trocar fundos completamente.

---

**CONTEÚDO PARA O ALUNO**

**1. Pipeline Profissional**

1. **Brief:** Definir objetivo, público, estilo, dimensões
2. **Prompting:** Criar prompt base com estilo e qualidade
3. **Generation:** Gerar 4-8 variações
4. **Selection:** Escolher melhores resultados

7. **Export:** Formatar para canal de destino

**2. Gestão de Prompts**

Organize prompts em biblioteca: categorias (produto, social, editorial), templates reutilizáveis, seeds documentados, histórico de resultados com notas.

**3. Otimização de Créditos**

Maximize resultado por token: teste em resolução baixa primeiro, refine prompt antes de gerar em alta qualidade, use seeds para variações controladas, batch generation para séries.
