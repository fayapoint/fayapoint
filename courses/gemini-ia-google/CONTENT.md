Plataforma gratuita para experimentar e prototipar com Gemini: interface visual para testar prompts, ajustar parâmetros (temperature, top_p, max_tokens), upload de arquivos para testes multimodais, exportar código pronto para integração.

**2. API Gemini — Primeiros Passos**

```python
import google.generativeai as genai

genai.configure(api_key="sua-api-key")
model = genai.GenerativeModel('gemini-pro')

response = model.generate_content("Explique quantum computing em 3 parágrafos")
print(response.text)
```

**3. Multimodal via API**

```python
import PIL.Image

model = genai.GenerativeModel('gemini-pro-vision')
img = PIL.Image.open('diagrama.png')
response = model.generate_content(["Explique este diagrama:", img])
print(response.text)
```

**1. Vertex AI — Enterprise Grade**

Para produção empresarial: SLAs garantidos, grounding com Google Search, integração com Google Cloud, fine-tuning de modelos, monitoramento e logging, compliance e privacidade.

**2. Grounding — Fatos Verificáveis**

O grounding conecta o Gemini ao Google Search para respostas baseadas em fatos atualizados. Reduz alucinações dramaticamente. Ideal para: chatbots de atendimento, assistentes de pesquisa, geradores de conteúdo factual.

**3. Safety Settings**

Configure filtros de segurança: categorias (harassment, hate speech, explicit content), níveis (block none, block few, block some, block most). Ajuste para seu caso de uso.

**4. RAG com Vertex AI**

Retrieval-Augmented Generation empresarial: indexe seus documentos no Vertex AI Search, Gemini consulta sua base de conhecimento para respostas contextualizadas. Perfeito para: bases de conhecimento internas, suporte ao cliente, documentação técnica.
